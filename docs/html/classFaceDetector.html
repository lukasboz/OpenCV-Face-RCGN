Detect faces in a given grayscale image.

Parameters
grayFrameGrayscale image in which to detect faces.
Returns
A vector of rectangles representing the detected faces.

This method takes a grayscale image as input and uses the loaded Haar Cascade classifier to detect faces. It returns a vector of rectangles, each representing the bounding box of a detected face.
