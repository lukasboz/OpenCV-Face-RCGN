◆ updateCountdown

void PinWindow::updateCountdown ( )
privateslot

Updates the countdown timer every second and uses it to update the text in the popup. Upon reaching 0, closes and deletes the timer, deletes the message box, and resets attempt counter.

Updates the countdown timer and popup text during lockout.

Every time timer emits a timeout() signal, the countdown decrements and the text in the message box updates accordingly. Once the countdown reaches 0, the timer stops and gets deleted, the popup gets deleted, and the attempt counter restarts.

Parameters

Validates if the inputted password is correct, upon successful verification, the user is sent to the next screen otherwise the attempt counter increases and once it reaches 2, the user is locked out of any attempts for 30 sec. After the countdown is over, the attempt counter resets and the user is free to make more attempts.
