Updates the countdown timer every second and uses it to update the text in the popup. Upon reaching 0, closes and deletes the timer, deletes the message box, and resets attempt counter. 

Updates the countdown timer and popup text during lockout.

Every time timer emits a timeout() signal, the countdown decrements and the text in the message box updates accordingly. Once the countdown reaches 0, the timer stops and gets deleted, the popup gets deleted, and the attempt counter restarts. \Hypertarget{classPinWindow_abda09706e1c2d2fdf70611f6a99a9dc5}\label{classPinWindow_abda09706e1c2d2fdf70611f6a99a9dc5}
