\doxysubsection{Detailed Description}
This class manages the addition and deletion of face images for profiles within the dataset. 

This class allows for the addition and deletion of face images for a preexisting profile, or for new profiles in the dataset. The class uses the Qt GUI libraries to prompt the user for a folder within the dataset directory.
